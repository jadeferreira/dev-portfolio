## Sobre o projeto

Este portfólio começou em **HTML, CSS e JavaScript puro** e foi evoluído para **React**, mantendo a estrutura visual original da página e reorganizando a aplicação em componentes reutilizáveis.

Na versão atual, a interface utiliza **CSS Modules** para modularização dos estilos, mantendo a base visual do projeto e facilitando a manutenção da paleta, das seções e do comportamento responsivo.

## Funcionalidades

- Layout responsivo para mobile, tablet e desktop
- Navegação suave entre seções
- Animações de scroll com `IntersectionObserver`
- Formulário de contato integrado com `EmailJS`
- Links diretos para WhatsApp, GitHub e LinkedIn
- Organização da interface em componentes React

## Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **React**
- **CSS Modules**
- **Vite**
- **EmailJS**
- **Font Awesome**
- **Google Fonts** — Poppins e Unbounded

## Estrutura atual

- `src/` — componentes, hooks, dados e estilos da aplicação React
- `public/assets/` — imagens e currículo usados no portfólio
- `index.html` — entrada da aplicação
