# 🌐 Jade Ferreira — Portfólio Pessoal

> Portfólio desenvolvido para apresentar minha trajetória, habilidades e projetos como Desenvolvedora Fullstack.

Visite o portfólio: https://jadeferreira.github.io/dev-portfolio/ :)



## ✨ Sobre o Projeto

Este portfólio foi criado do zero com foco em design moderno, responsividade e experiência do usuário. Ele apresenta minhas experiências profissionais, habilidades técnicas, projetos desenvolvidos e formas de contato.



## 🚀 Funcionalidades

- 💜 Design moderno com tema escuro e identidade visual própria
- 📱 Layout totalmente responsivo (mobile, tablet e desktop)
- 🎞️ Animações de scroll suaves com Intersection Observer
- 📩 Formulário de contato funcional integrado com EmailJS
- 🔗 Linkagem direta para WhatsApp e redes sociais
- ⚓ Navegação suave entre seções via menu



## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3** — Flexbox, Grid, animações, media queries
- **JavaScript** — Intersection Observer, EmailJS
- **Font Awesome** — ícones
- **Google Fonts** — Poppins & Unbounded
- **EmailJS** — envio de e-mails sem backend
